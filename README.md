# `school21-RT`

## Compilation

```
% brew install llvm sdl2 czmq
% make
```

## Usage

```
# As application
% ./RT scenes/multiobject.oadyaml

# As client-server
% ./RT.server
% ./RT.client scenes/multiobject.oadyaml
```
